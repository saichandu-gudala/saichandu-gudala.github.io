# Gudala Saichandu Portfolio

A free portfolio website built with React, Vite, Tailwind CSS, and Lucide Icons. The content is based on the uploaded resume.

## Tech Stack

- Frontend: React + Vite
- Styling: Tailwind CSS
- Icons: Lucide Icons
- Hosting: GitHub Pages
- Free domain: `yourname.github.io`

## Run Locally

```bash
npm install
npm run dev
```

Then open the local URL printed by Vite.

## Build

```bash
npm run build
```

## Deploy to GitHub Pages

1. Create a GitHub repository named `yourname.github.io` for the free root domain.
2. Push this project to that repository.
3. Run:

```bash
npm run deploy
```

If you use a project repository instead of `yourname.github.io`, update the `base` value in `vite.config.js`.
